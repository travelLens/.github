# 🌍 TravelLens

**데이터로 더 똑똑한 여행, AI로 더 완벽한 추천**

TravelLens는 여행지의 인기, 위험도, 비용, 최신 트렌드를 세계지도에서 탐색하고,
실시간 데이터와 AI를 활용해 여행지 선택을 돕는 여행 인사이트 플랫폼입니다.

## ✨ What We Provide

* 🗺️ **지도 기반 여행 탐색**
  국가별 여행 정보, 여행경보, 인기도를 세계지도에서 직관적으로 확인할 수 있습니다.

* 📈 **실시간 여행 트렌드 분석**
  사용자 행동 데이터를 기반으로 인기 여행지를 분석하고 시각화합니다.

* 🔎 **여행 콘텐츠 검색**
  뉴스, 블로그, 환율, 항공권 등 흩어진 여행 정보를 국가 단위로 통합합니다.

* 🤖 **AI 여행 추천**
  검색 데이터와 사용자 관심 정보를 바탕으로 개인화된 여행 답변을 제공합니다.

* 💬 **커뮤니티와 개인화 기능**
  실시간 채팅, 관심 국가 관리, 마이페이지 기능을 제공합니다.
